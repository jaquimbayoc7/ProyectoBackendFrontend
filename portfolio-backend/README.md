## Descripción

[Nest](https://github.com/nestjs/nest) es un repositorio inicial de TypeScript para el framework NestJS.

## Configuración del proyecto

```bash
$ npm install
```
## Compilar y ejecutar el proyecto
```bash
# desarrollo
$ npm run start

# modo observador
$ npm run start:dev

# modo producción
$ npm run start:prod
```
## Ejecutar Pruebas
```bash
# pruebas unitarias
$ npm run test

# pruebas e2e
$ npm run test:e2e

# cobertura de pruebas
$ npm run test:cov
```
